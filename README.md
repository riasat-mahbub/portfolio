## **Getting Started** 
1. Clone the repository:  
```bash
git clone https://github.com/riasat-mahbub/portfolio
```
2. Install dependencies:
```bash  
npm install
```
3. Start the development server:
```bash  
npm run dev
```
4. Open [http://localhost:4321](http://localhost:4321) in your browser
